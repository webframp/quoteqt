How to Get Session Token & Channel ID

  1. Open nightbot.tv and log in
  2. Open browser DevTools (F12) → Network tab
  3. Navigate to a channel you manage
  4. Look for requests to api.nightbot.tv
  5. Find the Authorization header: Session <token> - copy just the token part
  6. Find the Nightbot-Channel header - this is the Channel ID

Note: Session tokens appear to be long-lived, but if sync fails with auth errors, you'll need to update the token.
