To get your Nightbot access token:

  1. Go to nightbot.tv/dashboard and log in
  2. Drag this bookmarklet to your bookmarks bar: Get Nightbot Token
  3. Click the bookmarklet while on the Nightbot dashboard
  4. Paste the token below
Manual method (if bookmarklet doesn't work)
  1. Open browser Developer Tools (F12) â†’ Network tab
  2. Refresh the page or click on any menu item
  3. Find any request to api.nightbot.tv
  4. Look in Request Headers for Authorization: Bearer ...
  5. Copy the token (everything after "Bearer ")

Note: Tokens expire after ~30 days. You'll need to get a new one when it expires.

About

This tool allows you to:

  • Export - Download all your custom Nightbot commands as a JSON file
  • Import - Restore commands from a previously exported backup

Useful for:

  • Backing up commands before making major changes
  • Migrating commands to a new channel
  • Sharing command setups between channels
