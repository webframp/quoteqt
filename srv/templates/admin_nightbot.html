Export Commands (Bookmarklet)

Since Nightbot doesn't allow third-party API access, use this bookmarklet to export your commands directly:

  1. Drag this to your bookmarks bar: Export Nightbot Commands
  2. Go to nightbot.tv/commands and log in
  3. Click the bookmarklet - it will download a JSON file
  4. Upload that file below to import into another channel

Note: The bookmarklet extracts basic command info (name, message). Cooldown and user level default to 5s and "everyone".

About

This tool helps you backup and manage Nightbot commands:

  • Export - Use the bookmarklet above to download commands from Nightbot
  • Validate - Check backup files for errors before manual import

Useful for:

  • Backing up commands before making major changes
  • Sharing command setups between channels
  • Keeping an offline record of your commands

Note: Due to Nightbot API limitations, automatic import is not available. Commands must be added manually through the Nightbot dashboard.
