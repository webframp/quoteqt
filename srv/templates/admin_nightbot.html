About

This tool allows you to:

  • Export - Download all your custom Nightbot commands as a JSON file
  • Import - Restore commands from a previously exported backup

Useful for:

  • Backing up commands before making major changes
  • Migrating commands to a new channel
  • Sharing command setups between channels
