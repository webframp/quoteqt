Go Template Project

This is a starter template for building Go web applications on exe.dev. Customize this page and the code to create your own service.

This VM has a persistent disk, sudo, HTTPS, and SSH all wired up.

What next?

Customize this template

Edit the code to build your application.

  • Source code is in ~
  • Edit via SSH or open in VS Code
  • Chat with our coding agent, Shelley
  • sudo systemctl restart srv to pick up changes

What's included

This template provides the essentials.

  • HTTP server with routing and templates
  • SQLite database with migrations
  • Authentication via exe.dev headers
  • Systemd service configuration
